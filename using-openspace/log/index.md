# OpenSpace Log

:::{warning}
This section is incomplete and more of a placeholder page at the moment.
:::


The OpenSpace Log is located at {menuselection}`OpenSpace/logs/log.html`. Open this page in a browser and it will reveal a list of all commands executed by OpenSpace in the order they were executed. The most recent is a the bottom of the page.

The vast majority of these lines report successfully run commands. But, if there are errors or warnings, they will appear highlighted in red or yellow, respectively. And, the presence of errors are noted at the top of the page by listing the number of errors and warnings. The link will take you to the first error or warning on the page.
