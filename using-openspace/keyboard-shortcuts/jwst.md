# James Webb Space Telescope Profile Keyboard Shortcuts
Shortcuts found in the [James Webb Space Telescope profile](/profiles/missions/jwst/index), found at `data/profiles/missions/jwst.profile`.

(default-shortcuts)=
| Shortcut | Function |
| -------- | -------- |
| {kbd}`s` | Toggle satellite trails |
| {kbd}`p` | Toggle the Lagrangian Points |
| {kbd}`u` | `os.jwst.ToggleHudf` |
| {kbd}`o` | Toggle L2 Lagrangian Point |
| {kbd}`v` | `os.jwst.ToggleFov` |
| {kbd}`j` | `os.jwst.SetupLaunch` |
| {kbd}`k` | `os.jwst.ToggleSunTrail` |
| {kbd}`m` | `os.jwst.PlayForwards` |
| {kbd}`n` | `os.jwst.PlayBackwards` |
| {kbd}`b` | `os.jwst.ClearTimelapse` |
| {kbd}`g` | `os.jwst.ToggleTrailsExceptMoon` |
| {kbd}`y` | `os.jwst.ToggleDirection` |
| {kbd}`t` | `os.jwst.ToggleJwstTrails` |
