# Toolbar


![Toolbar Panels](/using-openspace/panels.png)

The Toolbar appears at the bottom of the OpenSpace window and gives you quick access to the panels. These panels can help with data manipulation, navigation, time control, and many other OpenSpace functions.


:::{warning}
This section is in progress and is, therefore, incomplete.
:::

:::{note}
The toolbar above shows all the possible options from the Top Bar's Windows menu. When you start OpenSpace, the Toolbar will look abbreviated---not all panels are present in the Toolbar by default.
:::


{.glossary .index-list}
[Scene Panel](/using-openspace/toolbar/scene/index)
: Manipulate data.

[Navigation Panel](/using-openspace/toolbar/navigation/index)
: Focus and fly.

[Time Panel](/using-openspace/toolbar/time/index)
: Manipulate the date, time, and the simulation speed.

[Actions Panel](/using-openspace/toolbar/actions/index)
: Automate scene adjustments.

[Record Panel](/using-openspace/toolbar/record/index)
: Save and playback a session.

[GeoLocation Panel](/using-openspace/toolbar/geolocation/index)
: Pinpoint and fly to a place.
