# OpenSpace API
OpenSpace provides two APIs for interacting with a running OpenSpace instance from external applications or web-based user interfaces. The [JavaScript / TypeScript API](https://github.com/openspace/openspace-api-js) is primarily intended for building custom web UIs and browser-based tools, while the [Python API](https://github.com/openspace/openspace-api-python) is suited for scripting, automation, and data-driven workflows.
