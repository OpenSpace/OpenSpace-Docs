# About


OpenSpace is an open-source, interactive data visualization software designed to visualize the entire known universe and portray our ongoing efforts to investigate the cosmos covering all possible scales.




Information about the different versions of OpenSpace, such as changelogs between versions or academic publications that came out of the project.
